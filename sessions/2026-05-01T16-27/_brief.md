# 📋 작업 브리프

**원 명령:** 이제 어던걸 기확해야할까?

## 요약
API 스키마와 기술적 구조가 정의되었으므로, 이제 실제 사용자가 경험할 서비스의 핵심 기능(MVP)을 정의하고, 이를 시장에 알리기 위한 비즈니스 및 마케팅 실행 계획을 수립해야 합니다.

## 분배
- **🔍 Researcher**: AI 뷰티/헤어 스타일링 서비스 경쟁사들을 분석하여, 사용자들이 가장 열광하는 기능과 현재 시장의 트렌드(예: 가상 헤어 피팅, 퍼스널 컬러 연동 등)를 조사하여 보고하세요.
- **💰 Business**: 정의된 기술 스키마를 바탕으로 MVP(최소 기능 제품)에 포함될 필수 기능 범위를 확정하고, 초기 사용자 확보를 위한 수익 모델(예: 프리미엄 리포트 결제, 제휴 미용실 연계 등)을 설계하세요.
- **✍️ Writer**: 사용자가 앱에 접속해 데이터를 입력하고 리포트를 받기까지의 '사용자 여정(User Journey)' 시나리오를 작성하고, 브랜드 톤(친절한 미용사)에 맞는 서비스 소개 카피를 제작하세요.
- **🎨 Designer**: 정의된 API 응답 데이터(StyleAnalysisResponse)를 기반으로, 사용자가 자신의 스타일을 한눈에 파악할 수 있는 '스타일 DNA 리포트'의 UI/UX 와이어프레임과 시각적 컨셉을 설계하세요.
- **📷 Instagram**: 서비스 런칭 전 기대감을 조성하기 위한 인스타그램 티저 캠페인(Reels 및 피드)의 콘텐츠 테마와 주차별 게시 스케줄을 기획하세요.
