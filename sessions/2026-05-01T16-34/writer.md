## 1. 🎬 릴스(Reels) 스크립트 (2종)

### [안 1] 공감 및 문제 해결형 (Short & Punchy)
*   **목표:** 실패 경험을 건드려 서비스의 필요성 각인
*   **후크(Hook):** "아직도 미용실 가서 '알아서 해주세요'라고 하세요? (그게 제일 위험해요!)"

| 구분 | 영상 화면 (Visual) | 오디오/자막 (Audio/Text) |
| :---립 | 미용실 의자에 앉아 불안한 표정으로 거울을 보는 모습 (흑백 처리) | "아직도 미용실 가서 '알아서 해주세요'라고 하세요?" |
| **본문** | '망한 머리' 예시 사진들이 빠르게 지나감 (빠른 템포의 비트) | "결과는 늘 실패, 매번 똑같은 고민... 이제 그만할 때 됐잖아요." |
| **반전** | 화면이 컬러로 전환되며 미드미 앱 진단 화면 등장 (밝고 경쾌함) | "드디어 도착했습니다. AI가 찾아주는 당신의 인생 스타일, 미드​미!" |
| **결과** | 진단 후 완성된 세련된 헤어스타일 모델 영상 | "얼굴형, 피부톤, 모질까지 분석해서 딱 맞는 스타일을 알려드려요." |
| **CTA** | 프로필 링크 클릭을 유도하는 손가락 모션 | "지금 프로필 링크에서 1분 만에 무료 진단받으세요! 👇" |

### [안 2] 비주얼 변신형 (Transformation)
*   **목표:** 미드미 사용 전/후의 극적인 변화를 보여주어 호기심 유도
*   **후크(Hook):** "내 얼굴형에 딱 맞는 머리, 1분 만에 찾는 법 🔍"

| 구분 | 영상 화면 (Visual) | 오디오/자막 (Text) |
| :--- | :--- | :--- |
| **후크** | 거울 앞에서 머리를 만지며 고민하는 사용자 클로즈업 | "내 얼굴형에 딱 맞는 머리, 1분 만에 찾는 법" |
| **본문** | 미드미 앱에 사진을 업로드하고 분석 게이지가 올라가는 화면 | "사진 한 장이면 끝! AI가 분석하는 스타일 DNA" |
| **결과** | 분석 결과 리포트(슬라이드) → 해당 스타일로 변신한 모습 | "찾았다, 내 인생 머리! ✨" |
| **CTA** | 앱 실행 화면과 함께 텍ext 강조 | "나만의 스타일 리포트 받기 (무료) 🔗" |
