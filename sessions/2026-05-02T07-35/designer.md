확립된 시각 시스템(Rose Tone, Glassmorphism, Serif)을 기반으로 '웹사이트 진단 결과 페이지'와 '어드민 대시보드'의 최종 와이어프레임을 구체적인 디자인 브리프 형태로 제출합니다.

## 1. 웹사이트 진단 결과 페이지 (Website Diagnosis Result Page) 디자인 브리프

이 페이지는 사용자에게 자신의 스타일 분석 결과를 친절하고 신뢰감 있게 전달하여 다음 단계(가입 유도)로의 전환을 극대화하는 것을 목표로 합니다. 로즈 톤과 글래스모피즘을 적용하여 따뜻하면서도 전문적인 느낌을 동시에 확보합니다.

**🎨 디자인 컨셉:** Warm Clarity (따뜻함과 명확성의 조화)
**🖼️ 레이아웃:** 중앙 집중형 카드(Card-based) 및 계층적 정보 구조.

### 1.1. 컬러 및 타이포그래피 시스템 적용

*   **배경색 (Background):** `#FCE4EC` (매우 연한 로즈 톤, 부드러운 배경 제공)
*   **카드/패널 색상 (Glassmorphism Base):** `background-color: rgba(255, 255, 255, 0.6);` (반투명 백색)
    *   **Border:** `1px solid rgba(255, 122, 85, 0.3)` (로즈 톤의 은은한 테두리)
    *   **Blur Effect:** `backdrop-filter: blur(16px);` (Glassmorphism 구현)
*   **주요 타이포그래피 (Serif):** `font-family: 'Playfair Display', serif;` (제목 및 핵심 지표에 적용)
*   **보조 타이포그래피:** `font-family: 'Noto Sans KR', sans-serif;` (본문 설명에 적용, 가독성 확보)
*   **핵심 액션 컬러 (Accent):** `#E07A5F` (로즈 톤 메인 컬러)

### 1.2. 섹션별 상세 와이어프레임

| 영역 | 구성 요소 | 디자인 지침 | 데이터 시각화 예시 |
| :--- | :--- | :--- | :--- |
| **Header** | 로고 및 제목 | 상단 중앙 배치. Serif 폰트로 브랜드명 강조. | - |
| **Diagnosis Score (핵심)** | 나에게 맞는 스타일 점수 | 가장 큰 폰트(H1/H2). 로즈 톤 그라데이션 배경 위에 배치. | `92%` (큰 숫자) + "당신의 스타일 DNA" |
| **Style Insight** | 개인화된 분석 결과 | Glassmorphism 카드 내부에 배치. 친절한 말투의 설명 문구 사용. | AI가 추천하는 스타일 이름 (예: '로맨틱 보헤미안') |
| **Data Visualization** | 스타일 DNA 상세 분석 | 3~4개의 핵심 요소(얼굴형, 분위기, 선호 컬러 등)를 아이콘과 함께 배치. | 원형 차트 또는 도넛 차트를 사용하여 각 요소의 중요도 시각화. |
| **Actionable Steps** | 다음 단계 제안 | 구체적인 개선 방법 제시. CTA 버튼은 Accent 색상(`#E07A5F`)으로 강조. | "더 깊은 분석하기" (Primary CTA) |

**✨ 최종 시각적 특징:** 전체적으로 부드러운 로즈 톤 배경 위에 투명하고 은은한 카드들이 떠 있는 듯한 느낌(Glassmorphism)을 주어, 복잡한 데이터가 아닌 '나에 대한 친절한 발견'이라는 느낌을 전달합니다.

## 2. 어드민 대시보드 (Admin Dashboard) 디자인 브리프

이 대시보드는 LTV 측정 및 퍼널 분석 등 전문적인 데이터를 다루므로, Glassmorphism의 투명함과 Serif 타이포그래피의 권위를 결합하여 정보의 가독성과 신뢰도를 최우선으로 확보합니다.

**🎨 디자인 컨셉:** Professional Clarity (전문성과 명확성의 조화)
**🖼️ 레이아웃:** 그리드 기반(Grid-based) 대시보드, 좌측 내비게이션 바와 우측 메인 콘텐츠 영역 분리.

### 2.1. 컬러 및 타이포그래피 시스템 적용

*   **배경색 (Background):** `#F8F4F9` (약간 더 차분하고 전문적인 느낌의 밝은 회갈색 계열)
*   **카드/패널 색상 (Glassmorphism Base):** `background-color: rgba(255, 255, 255, 0.75);` (조금 더 불투명하게 설정하여 데이터의 선명도 확보)
    *   **Border:** `1px solid rgba(224, 122, 85, 0.4)` (로즈 톤을 은은하게 사용한 경계선)
    *   **Blur Effect:** `backdrop-filter: blur(20px);` (깊이감을 강조하는 블러 효과)
*   **주요 타이포그래피 (Serif):** `font-family: 'Cormorant Garamond', serif;` (데이터 라벨 및 핵심 지표에 적용하여 권위 부여)
*   **보조 타이포그래피:** `font-family: 'Roboto', sans-serif;` (차트 값, 상세 설명 텍스트에 적용)
*   **KPI 색상 할당:** 성공/성장 지표는 `#4CAF50` (녹색 계열), 주의/이탈 지표는 `#FF9800` (주황 계열), 위험 지표는 `#F44336` (빨간색 계열)를 사용합니다.

### 2.2. 섹션별 상세 와이어프레임

| 영역 | 구성 요소 | 디자인 지침 | 데이터 시각화 예시 |
| :--- | :--- | :--- | :--- |
| **Sidebar Navigation** | 메뉴 및 상태 표시 | Glassmorphism 스타일로 제작하여, 현재 활성 메뉴는 로즈 톤으로 강조. | LTV 추이, 구독 상태(Enum) 표시. |
| **KPI Summary (상단)** | 핵심 성과 지표 요약 | 4~6개의 주요 KPI를 카드 형태로 배치. 각 카드 상단에 색상 코드를 명확히 표시. | `LTV: $1,200`, `Churn Rate: 5%` (Serif와 산세리프 혼용) |
| **Funnel Analysis** | 고객 여정 분석 | Acquisition_Source -> Trial -> Paid 등으로의 흐름을 시각화. | 단계별 퍼널 플로우 차트. 이탈 지점(`Churn_Point`)에 주황색 강조 표시. |
| **Data Table (Detail)** | 세부 데이터 목록 | `Acquisition_Source`, `Churn_Point` 등의 데이터를 테이블 형태로 제시. | 행(Row) 단위로 Glassmorphism 카드 적용. 데이터 셀은 명확한 대비를 유지. |

**✨ 최종 시각적 특징:** 어드민 페이지는 차분하고 정돈된 느낌을 주되, Glassmorphism의 투명함을 통해 데이터가 '숨겨진 정보'가 아닌 '명확하게 드러난 사실'로 느껴지게 합니다. Serif 타이포그래피는 전문적인 분석 결과를 전달하는 데 집중합니다.
