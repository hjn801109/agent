## 웹사이트 메인 랜딩 페이지 초기 디자인 시안 (핵심 섹션)

CEO의 지시와 확립된 브랜드 가이드라인(Rose 톤, 세리프 타이포그래피, 글래스모피즘)을 적용하여 웹사이트의 핵심 인상을 결정할 수 있는 메인 섹션의 디자인 시안을 구체적으로 제시합니다. 이 시안은 따뜻함과 전문성을 동시에 전달하는 데 중점을 둡니다.

### 1. 디자인 시스템 요약 (재확인)

| 요소 | 적용 스타일 | 상세 정의 |
| :--- | :--- | :--- |
| **브랜드 컬러** | Rose Tone 기반 | Primary: `#E07A5F` (따뜻함/포인트), Secondary: `#F4ACB7` (배경/보조 톤), Neutral: `#F8F6F2` (밝은 배경) |
| **타이포그래피** | Serif 적용 | Headline: Playfair Display (강한 존재감), Body: Lora 또는 Merriweather (가독성 및 친근함) |
| **레이아웃/효과** | Glassmorphism | 반투명한 블러 효과와 미세한 그림자(Soft Shadow)를 사용하여 깊이감을 부여 |

### 2. 핵심 섹션 디자인 시안: Hero Section

이 섹션은 방문자가 페이지에 접속했을 때 가장 먼저 인식해야 할 '신뢰감'과 '친근함'을 동시에 전달하는 것을 목표로 합니다.

**섹션 제목:** Style DNA 리포트 (가장 중요한 CTA 유도)

**레이아웃 좌표 및 구성:** 전체 너비 100%를 사용하며, 중앙 정렬을 기본으로 합니다.

| 영역 | 크기/위치 | 디자인 요소 상세 설명 |
| :--- | :--- | :--- |
| **배경 컨테이너** | Full Width (min-height: 70vh) | 배경색: `#F8F6F2` (매우 밝은 로즈 베이지). Glassmorphism 적용. 블러(Blur Radius): 16px. 배경에 미세한 Rose Tone 그라데이션을 은은하게 삽입하여 따뜻함을 부여. |
| **헤드라인 (H1)** | 중앙 상단 (Padding: 5% 상하) | **폰트:** Playfair Display, Bold. **색상:** `#3D405B` (짙은 차콜). 크기: 68px. *내용 예시: "당신의 숨겨진 스타일 DNA를 발견하세요"* |
| **서브 헤드라인** | H1 바로 아래 | **폰트:** Lora, Regular. **색상:** `#E07A5F` (Rose Tone). 크기: 24px. *내용 예시: "AI가 분석한 나에게 가장 잘 어울리는 헤어스타일과 스타일링 솔루션"* |
| **핵심 CTA 버튼** | 중앙 하단 배치 | **스타일:** Glassmorphism 카드 형태. 배경색: 반투명 Rose Tone (`rgba(224, 122, 95, 0.15)`). 테두리: 1px Solid `#E07A5F`. 버튼 내부 텍스트는 흰색(`#3D405B`)으로 명확하게 표시. |
| **보조 요소** | CTA 아래 우측 | 작은 아이콘 또는 인용구 박스. 배경은 불투명도를 낮춘 연한 Rose Tone (`#F4ACB7`). |

### 3. 섹션 상세 디자인 가이드 (Glassmorphism 적용 예시)

**목표:** 데이터와 정보를 시각적으로 '투명하게' 제시하여 신뢰감을 확보합니다.

1.  **카드 형태 정의:** 모든 정보 블록(예: 서비스 소개, 진단 결과 요약)은 **Glassmorphism 카드** 형태로 구현합니다.
    *   `backdrop-filter: blur(16px);` (필수 적용)
    *   `background-color: rgba(255, 255, 255, 0.8);` (약간의 불투명도 유지)
    *   `border: 1px solid rgba(224, 122, 95, 0.3);` (Rose Tone을 활용한 은은한 테두리)

2.  **타이포그래피 계층:**
    *   가장 중요한 정보는 **Serif체 (Playfair Display)**로 처리하여 클래식하고 전문적인 느낌을 부여합니다.
    *   세부 설명이나 데이터 포인트는 **Lora/Merriweather**를 사용하여 친절함과 가독성을 높입니다.

3.  **컬러 사용 전략:**
    *   **Primary Action Color (`#E07A5F`):** CTA 버튼, 핵심 강조 문구에만 제한적으로 사용하여 시선을 집중시킵니다.
    *   **Background/Depth Color (`#F4ACB7`, `#F8F6F2`):** 배경과 카드 컨테이너에 사용되어 Rose Tone의 따뜻함과 Glassmorphism의 은은한 깊이를 동시에 표현합니다.
