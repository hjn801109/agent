# 🔍 AI 추천 엔진 최적화 통합 데이터 스키마 (Master Draft)

미드미(MIDMI)와 같은 뷰티/헤어 플랫폼에서 단일 사용자(User)를 기준으로 모든 정적/동적 데이터를 한 번에 조회하고 관리하기 위한 통합 JSON Document 스키마입니다. NoSQL 데이터베이스 환경에 최적화되어 있습니다.
