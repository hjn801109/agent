# 📱 미드미 인스타그램 콘텐츠 전략: "Confidence Finder"

## 1. 콘텐츠 핵심 전략 (3-Pillar Strategy)
모든 콘텐츠는 **[발견(Reels) → 학습(Carousel) → 유입(Link in Bio)]**의 흐름을 따릅니다.

| 구분 | 콘텐츠 유형 | 목적 | 핵심 내용 |
| :--- | :--- | :--- | :--- |
| **Pillar 1: The Magic** | **Reels (숏폼)** | **도달 & 발견** | AI가 얼굴형을 분석하고 스타일을 변환하는 'Before & After' 영상 (시각적 충격) |
| **리액션 중심** | | | "이게 나라고?" 하는 반응을 유도하는 빠른 템포의 편집 |
| **Pillar 2: The Guide** | **Carousel (카드뉴스)** | **저장 & 신뢰** | 얼굴형별(둥근형, 긴형, 각진형) 실패 없는 헤어 가이드 |
| **정보 제공 중심** | | | "이 게시물 저장해두고 미용실 갈 때 보여주세요"라는 문구 활용 |
| **Pillar 3: The Pro** | **Stories / Reels** | **소통 & 권위** | 헤어 전문가들의 팁, 최신 트렌드 리포트, AI 기능 업데이트 소식 |
| **커뮤니티 중심** | | | 투표 기능(A vs B 스타일)을 통한 유저 참여 유도 |

## 2. 주간 콘텐츠 실행 계획 (Sample Weekly Plan)

| 요일 | 콘텐츠 형태 | 주제 (Topic) | 비즈니스 목표 (KPI) |
| :--- | :--- | :--- | :--- |
| **월** | **Reels** | AI로 10초 만에 인생 머리 찾기 (Process) | 도달률 (Reach) |
| **수** | **Carousel** | 둥근 얼굴형을 길어 보이게 만드는 레이어드 컷 가이드 | 저장률 (Save) |
| **금** | **Reels** | [Before/After] 덥수룩한 머리 $\rightarrow$ AI 추천 스타일 적용 | 공유률 (Share) |
| **일** | **Stories** | 이번 주 트렌드 헤어 투표 (A 스타일 vs B 스타일) | 참여율 (Engagement) |

## 3. 수익화 및 유입 설계 (Conversion Path)
*   **Bio (프로필):** "내 얼굴형에 맞는 헤어스타일, AI로 10초 만에 확인하기 👇 [Link]"
*   **CTA (Call to Action):** 모든 게시물 마지막 슬라이드/캡션에 **"지금 프로필 링크에서 무료로 분석해보세요"** 문구 삽등.
*   **광고 연계:** 반응이 좋은(저장/공유가 높은) 카드뉴스를 타겟 광고(헤어/뷰티 관심사)로 전환하여 유료 유입 극대화.
